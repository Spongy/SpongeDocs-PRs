Comparing Bukkit to SpongeÂ¶

While the BukkitAPI is notably different to SpongAPI, there are some Events and classes which have an equivalent. This page is intended to provide a (probably incomplete) list to help former Bukkit devs out when first trying SpongeAPI.

Note

To keep the table small, org.bukkit. and org.spongepowered.api. were removed in front of the class names.

We start with Bukkit 1.7. The additions the Spigot Team made to the API for 1.8 are in a separate table below.
